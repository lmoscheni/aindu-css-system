Layout System

Row

How you can see on this example of a row, any row of the grid system has 12 columns in three screen sizes (sm, md and lg), you can decide how many spaces of the twelve available you want to use in any column on of the three screen sizes.

In this example I use for big and middle screens sizes 1 space for every column, but for small screens I choose 2 space for every column. As result the column class is col -sm-2 -md-1 -lg-1.
